 Core Project:
 
 The Core project holds the Core or Common components that are used by 
 most if not all other projects.  The main objective of these classes is to 
 provide basic functionality such as property management and logging with a minimum 
 of external dependencies.
 
 
 Requires Java 9 or greater
 Dependencies:  
 	Testing: log4j 2.13.2 or greater
 	Compile: None
 	Runtime: None
